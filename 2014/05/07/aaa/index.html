hexo搭建的静态博客

前言

使用hexo搭建的静态博客。。

这两天也是折腾死了，

其实到最后并不是很难，主要是github和git的使用之前不是很熟,git的问题查了很多网站，然而最后还是没有解决ssh传输的问题
于是每次更新都要输入用户密码。。。回来再研究一下吧

—-待更—-

多说评论

hexo的自带的评论据说是Disqus，大家的评价也很高。但是由于。。众所周知的不可说之墙，找了国内的多说

多说

也有说多说评价一般般的。。总之先用着吧

找到“创建站点获取通用代码”，创建站点，记住你的short_name，就是“站点名称”

然后根据教程 Hexo使用多说教程

在 _config.yml 中添加多说的配置：

duoshuo_shortname: 你站点的short_name

修改 themes\landscape\layout\_partial\article.ejs 模板

把

<% if (!index && post.comments && config.disqus_shortname){ %>
<section id="comments">
  <div id="disqus_thread">
    <noscript>Please enable JavaScript to view the <a href="//disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
  </div>
</section>
<% } %>

改为

<% if (!index && post.comments && config.duoshuo_shortname){ %>
<section id="comments">
  <!-- 多说评论框 start -->
  <div class="ds-thread" data-thread-key="<%= post.layout %>-<%= post.slug %>" data-title="<%= post.title %>" data-url="<%= page.permalink %>"></div>
  <!-- 多说评论框 end -->
  <!-- 多说公共JS代码 start (一个网页只需插入一次) -->
  <script type="text/javascript">
  var duoshuoQuery = {short_name:'<%= config.duoshuo_shortname %>'};
    (function() {
      var ds = document.createElement('script');
      ds.type = 'text/javascript';ds.async = true;
      ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
      ds.charset = 'UTF-8';
      (document.getElementsByTagName('head')[0] 
       || document.getElementsByTagName('body')[0]).appendChild(ds);
    })();
    </script>
  <!-- 多说公共JS代码 end -->
</section>
<% } %>
